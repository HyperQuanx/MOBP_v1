TreeSet<E> 컬렉션

- Set<E>의 기본 메서드 이외의 검색 관련 메서드

메서드 이름 리턴타입 기능
first() E Set 원소들 중에서 가장 작은 원소 값 리턴
last() E Set 원소들 중에서 가장 큰 원소 값 리턴
lower(E) E 매개변수로 입력된 원소보다 작은 수 중 가장 큰 수
higher(E) E 매개변수로 입력된 원소보다 큰 수 중 가장 작은 수
floor(E) E 매개변수로 입력된 원소보다 작은 수 중 같거나 가장 큰 수
ceiling(E) E 매개변수로 입력된 원소보다 큰 수 중 같거나 가장 작은 수
pollFirst() E 원소들 중에서 가장 작은 원소 값 꺼내어 리턴
pollLast() E 원소들 중에서 가장 큰 원소 값 리턴
headSet(E) SortedSet<E> 매개변수보다 작은 모든 원소들로 구성된 Set 리턴
(default: 매개변수 값 제외)
headSet(E, boolean inclusive) NavigableSet<E> 첫 번째 매개변수보다 작은 모든 원소들로 구성된 Set 리턴,
두 번째 값에 따라 매개변수 값 포함 여부 결정
tailSet(E) SortedSet<E> 매개변수보다 큰 모든 원소의 집합 Set 리턴
(default: 매개변수 포함)
tailSet(E, boolean inclusive) NavigableSet<E> 매개변수보다 큰 모든 원소의 집합 Set 리턴
(두 번째 매개변수에 따라 매개변수 포함 여부 결정)
subSet(E from, E to) SortedSet<E> 첫 번째보다 같거나 크고, 두 번째보다 작은 원소의 집합 Set을 리턴
(from 포함, to 미포함)
subSet(E from, boolean inclusive, E to, boolean inclusive) NavigableSet<E> 첫 번째보다 크고 세 번째보다 작은 원소의 집합 Set 리턴
(두 번째와 네 번째 값에 따라 포함 여부 결정)
descendingSet() NavigableSet<E> 내림차순의 의미가 아님, 현재 정렬 기준의 반대로 변환

Map<K, V> 컬렉션의 주요 메서드

메서드 이름 리턴타입 기능
put(K, V) V 입력 매개변수(키, 값) 을 Map 객체에 추가
putAll(Map<? extends K, ? extends V> M) void 입력 매개변수의 Map 객체를 통째로 추가
replace(K, V) V Key에 해당하는 값을 value로 변경
(old 값을 리턴, 해당 Key가 존재하지 않으면 null 리턴)
replace(K, V old, V new) boolean Key에 해당하는 V old를 V new로 변경
get(Object Key) V 매개변수의 Key 값에 해당하는 value 리턴
containsKey(Object key) boolean 매개변수의 key 값이 포함되어 있는지 여부
containsValue(Object value) boolean 매개변수의 value 값이 포함되어 있는지 여부
keySet() Set<K> Map 데이터들 중에서 key만 뽑아서 Set 객체로 리턴
entrySet() Set<Entry<K,V>> Map의 각 Entry를 Set 객체로 담아 리턴
size() int Map에 포함된 데이터(Entry) 갯수
remove(Object key) V 입력 매개변수의 key를 갖는 엔트리 삭제
(단, key가 없으면 아무 동작 안함)
remove(Object key, Object value) boolean 입력 매개변수의 (key, value)를 갖는 엔트리를 삭제
(엔트리가 없으면 아무 동작 안함)
clear() void Map 객체 내의 모든 데이터를 삭제

TreeMap<K, V> 의 주요 메서드

메서드 이름 리턴타입 기능
firstKey() K TreeMap에서 가장 작은 Key를 리턴
firstEntry() Map.Entry<K, V> TreeMap에서 가장 작은 Key에 해당하는 Entry 리턴
lastKey() K TreeMap에서 가장 큰 Key를 리턴
lastEntry() Map.Entry<K, V> TreeMap에서 가장 큰 Key에 해당하는 Entry 리턴
lowerKey(K) K 지정된 Key보다 작은 Key 중에서 가장 큰 Key 리턴
lowerEntry(K) Map.Entry<K, V> 지정된 Key보다 작은 Key 중에서 가장 큰 Key에 해당하는 Entry 리턴
higherKey(K) K 지정된 Key보다 큰 Key 중에서 가장 작은 Key 리턴
higherEntry(K) Map.Entry<K, V> 지정된 Key보다 큰 Key 중에서 가장 작은 Key에 해당하는 Entry 리턴
pollFirstEntry() Map.Entry<K, V> 가장 작은 Key에 해당하는 Entry를 제거하고 리턴
pollLastEntry() Map.Entry<K, V> 가장 큰 Key에 해당하는 Entry를 제거하고 리턴
headMap(K) SortedMap<K, V> Key보다 작은 key 값을 가지는 모든 엔트리를 포함한 Map 리턴
(default: 매개변수 미포함)
headMap(K, boolean inclusive) NavigableMap<K, V> Key보다 작은 key 값을 가지는 모든 엔트리를 포함한 Map 리턴
(매개변수 포함 여부를 플래그로 설정)
tailMap(K) SortedMap<K, V> Key보다 큰 key 값을 가지는 모든 엔트리를 포함한 Map 리턴
(default: 매개변수 미포함)
tailMap(K, boolean inclusive) NavigableMap<K, V> Key보다 큰 key 값을 가지는 모든 엔트리를 포함한 Map 리턴
(매개변수 포함 여부를 플래그로 설정)
subMap(K from, K to) SortedMap<K, V> from 이상 to 미만의 엔트리로 구성된 Map 리턴
subMap(K from, boolean if, K to, boolean if) NavigableMap<K, V> from 이상 to 미만의 범위를 지정, 포함 여부는 플래그로 설정

Stack 클래스의 주요 메서드와 기능

메서드 이름 리턴타입 기능
push(E item) E 매개변수 item을 stack 추가
pop() E 최상위 데이터를 꺼냄(데이터 개수 줄어듬)
peek() E 가장 상위에 있는 요소 조회(데이터 갯수 변화 없음)
search(Object) int stack 원소의 위치값 리턴
(최상위 기준 1로 시작 아래로 내려 갈 수록 1씩 증가)
empty() boolean stack 객체가 비워져 있는지 여부 리턴
